# Quick Builder v2 - User Guide

## Overview
Quick Builder v2 helps you create high-converting landing pages in 3 simple steps.

## How to Access
1. Go to `/app/new-builder`
2. Click "কুইক বিল্ডার দিয়ে তৈরি করুন" (green button)

## Step-by-Step Guide

### Step 1: Select Your Intent
- **Product Type:** Single product or Multiple products
- **Goal:** Direct Sales (COD) or Lead Generation (WhatsApp)
- **Traffic Source:** Facebook Ads, TikTok, or Organic/Search

### Step 2: Connect Your Product
- Select from existing products, OR
- Create a new product inline (name, price, image)

### Step 3: Choose Template
- System suggests 3 best templates based on your intent
- Select one and click "ল্যান্ডিং পেইজ তৈরি করুন"

## After Creation
- Page opens in editor with optimized sections
- Click palette icon 🎨 to change section variants
- Edit content as needed

## Section Variants
| Section | Variants Available |
|---------|-------------------|
| Hero | Product-focused, Offer-focused, Video-focused, Text-focused |
| Testimonials | Cards, Carousel, Avatars, Screenshots, Star-rating |
| CTA | Button-only, With-trust, Urgency |
| Features | Grid-3, Grid-4, List |
| Social | Counter, Live-feed, Badges |

## Style Settings
Go to `/app/settings/landing` for:
- Brand color picker
- Button style (rounded/sharp/pill)
- Font family
- Dark mode toggle

## Checkout Settings
In `/app/settings/landing`:
- Toggle between Redirect and Modal checkout
- Modal checkout keeps customers on page

## Best Practices
- Facebook Ads → Use flash-sale template, urgency CTA
- TikTok → Use video-focus template, video hero variant
- Organic → Use premium-bd template, detailed sections
